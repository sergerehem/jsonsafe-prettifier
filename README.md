# JSONSafe Prettifier

Converte entradas em formato JSON/JSONSafe para JSON válido e formatado (prettify), com:

- validação de JSON
- destaque de erro com trecho/posição
- sugestão de correção
- syntax highlight na saída
- copiar/baixar resultado

## Rodar local

```bash
python3 -m http.server 8891
```

Abra: `http://localhost:8891`
